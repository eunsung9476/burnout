# 데이터 파일 위치

이 폴더에 아래 **두 개의 데이터 파일**을 넣어주세요. (파일 이름을 정확히 맞춰주세요)

```
data/final_school_data.csv      ← 메인 학교 데이터 (CSV)
data/school_wide_final.xlsx     ← 연도별 추이 데이터 (Excel)
```

- 파일을 넣은 뒤 페이지를 새로고침하면 백엔드 서버가 자동으로 읽어 대시보드에 반영됩니다.
- CSV 인코딩은 **UTF-8** 권장입니다. (한글이 깨지면 UTF-8 로 다시 저장 후 넣어주세요)
- 샘플 데이터는 생성하지 않습니다. 실제 파일을 넣어야 데이터가 표시됩니다.
